#### 规则变化

与过去分词的规则变化完全相同：

- **直接加 -ed**：walk → walked, talk → talked
- **以 -e 结尾加 -d**：smile → smiled, dance → danced
- **以辅音 + -y 结尾**：study → studied, cry → cried
- **[[重读闭音节双写辅音]]**：stop → stopped, plan → planned

####  不规则动词的过去式

不规则动词的过去式需[单独记忆](附录)，常见模式包括：

- **元音变化**：
    - 例：come → came, give → gave, run → ran
- **完全不变**：
    - 例：cut → cut, put → put, hit → hit
- **辅音变化或复杂变化**：
    - 例：buy → bought, go → went, have → had
- **与过去分词不同**：
    - 例：break → broke（过去式）→ broken（过去分词）
    - 例：drink → drank（过去式）→ drunk（过去分词）

####  特殊情况

- **双重形式**：同过去分词，部分动词有规则和不规则两种形式。
    - 例：spill → spilled/spilt, spoil → spoiled/spoilt
- **美式与英式差异**：
    - 例：learn → learned（美式）/learnt（英式）
- **[[发音]]**：过去式的发音规则与过去分词相同（/ɪd/, /t/, /d/）。
