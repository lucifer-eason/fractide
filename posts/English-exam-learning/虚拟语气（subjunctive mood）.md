### 一， 基本定义

>[!tip] 表示（虚拟）即不现实的动作/状态。
>虚拟的动作并不在真实世界发生，所以所有的虚拟动作都是产生于人类脑中的所以虚拟语气都带有==强烈的主观色彩==。

### 二，句式限制

>[!tip] 为了体现不真实（虚拟而非真实的动作），虚拟语气必须在从句（subjunctive sentence）中出现：
| 我遇见了外星人。             | 简单句:没有条件/解释，无法判断真实与否 | 简单句 |
| -------------------- | -------------------- | --- |
| ==我希望==遇见外星人。        | 主句解释了：我遇到外星人是我的愿望    | 复杂句 |
| ==如果我登上飞碟==，我会遇见外星人。 | 从句提供了：我遇到外星人的前提条件    | 复杂句 |
	这里是是指99%的情况，但在一些==语境==中虚拟语气也可以单独出现。

### 三，语法结构

#### A，添加条件的虚拟语气：

>[!结构] 结构：主句（假设的动作）+ 从句（条件）。

| 虚拟时态  | 条件从句(主要if为主)                                    |
| ----- | ----------------------------------------------- |
| 相对于现在 | 主语 + 动词过去式[^1]                                  |
| 相对于过去 | 主语 + had 动词过去分词                                 |
| 相对于未来 | 主语 + should / were to/ 动词过去式 ==[这里的区别下图由详细表格]== |
>[!tip] 这里的条件句从句的形式并没有限制（不是只有if引导）：
>Without air, there would be no living thins.  (under, with, without)
>Born in better times, he would have been a scholar. (分词短语)
>To hear him speak English, one would take him for a foreigner. (不定式)
>Another sound from me and the snake might strike. (名词短语)
>She eats too much everyday, or she would be thinner.（or/other wise 主句常为陈述语气）
>I would go abroad but I am short of money. (but/ but that  从句常为陈述语气) 

---

>[!tip] 虚拟语气的产生就是相对陈述语气产生区别，在条件从句中主要通过添加==情态助动词的过去形式==来区别（并不只有这一种）。


| 虚拟时态  | 主句谓语（陈述，方便对比）[^3] | 主句(虚拟)                                             |
| ----- | ----------------- | -------------------------------------------------- |
| 相对于现在 | 一般现在时             | 主语 + 情态动词 + 动词原形                                   |
| 相对于过去 | 动词过去式             | 主语 + 情态动词 + have + 动词过去分词                          |
| 相对于未来 | will + 动词原形       | 主语 + 情态动词 + 动词原形 / 或 直接使用祈使句（这里的条件句必须使用should[^2]） |

>[!tip] 以及不同情态动词的主要含义，这里根据文义语境灵活切换：

| 实际情态  | 虚拟形式（为区分陈述语气） | 常见含义         |
| ----- | ------------- | ------------ |
| will  | would         | 意愿、假设、预测的主观化 |
| can   | could         | 能力假设、可能性降低   |
| may   | might         | 可能性、猜测降低     |
| shall | should        | 建议、义务、推测     |
| be    | were          | 状态假设（所有人称）   |

---
一个相对明确的对比：

| 助动词       | 语气                | 含义               | 示例                                        |
| --------- | ----------------- | ---------------- | ----------------------------------------- |
| **will**  | 真实语气（indicative）  | 表示真实未来、计划或确定性    | It **will** stop raining soon.（雨马上会停）     |
| **would** | 虚拟语气（subjunctive） | 表示主观愿望、假设、不确定或不满 | I wish it **would** stop raining.（但现在还没停） |



>[!tip] 主句，从句的时态并没有固定搭配，可以任意组合
>EX: If I had finished the homework, I would play now.


---
### B， 除了提供条件外，虚拟语气可以使用在所有类型的[[复杂句complex sentence]](从句)中：

在这里 通常直接在陈述语气的基础上直接回溯一个时态（backshift）。

| 虚拟时态  | 主句谓语（陈述，方便对比）[^3] | 主句(虚拟)             |
| ----- | ----------------- | ------------------ |
| 相对于现在 | 一般现在时             | 主语 + 过去式           |
| 相对于过去 | 动词过去式             | 主语 + had + 动词过去分词  |
| 相对于未来 | will + 动词原形       | 主语 + should/could  |

>[!tip] 其他所有语法同样基于陈述语气下的从句。

[^1]在虚拟语气中，无论任何人称，规定使用be的过去式（were）： If I were you...
[^2]EX: If it shoud rain, don't expect me.
[^3]这里可以观察到，只是直接在陈述语气上强制加上助动词，以体现虚拟：EX: I met the alien. => I would have met the alien.




| 结构类型                   | 用于表达              | 条件句结构               | 主句结构                 | 举例                                                                |
| ---------------------- | ----------------- | ------------------- | -------------------- | ----------------------------------------------------------------- |
| ✅ **一般虚拟（表示未来可能性小）**   | 对未来的假设（可能发生，也可能不） | If + 过去式            | would / could + 动词原形 | If it **rained** tomorrow, we **would stay** home.（如果明天下雨，我们就待在家） |
| ✅ **更不可能的未来（were to）** | 几乎不可能发生           | If + were to + 动词原形 | would + 动词原形         | If I **were to win** the lottery, I **would travel** the world.   |
| ✅ **书面正式假设（should）**   | 表示假设“万一”或礼貌       | If + should + 动词原形  | would + 动词原形         | If you **should need** help, just call me.（万一你需要帮忙）               |
