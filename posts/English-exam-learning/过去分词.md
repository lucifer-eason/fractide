#### 规则变化

- **直接加 -ed**：适用于大多数动词。
    - 例：work → worked, play → played, visit → visited
- **以 -e 结尾的动词**：直接加 -d。
    - 例：love → loved, decide → decided
- **以辅音 + -y 结尾的动词**：将 -y 改为 -i，再加 -ed。
    - 例：study → studied, carry → carried
    - **例外**：以元音 + -y 结尾的动词直接加 -ed。
        - 例：play → played, enjoy → enjoyed
- **重读闭音节，单辅音结尾**：双写最后一个辅音字母，再加 -ed。
    - 例：stop → stopped, plan → planned, admit → admitted
    - **不双写的情况**：同现在分词，重音不在最后一音节或以 -w, -x, -y 结尾。
        - 例：listen → listened, fix → fixed

####  不规则动词的过去分词

不规则动词的过去分词形式不遵循规则，需[单独记忆](附录)。以下按常见模式分类：

- **过去式与过去分词相同**：
    - buy → bought → bought, 
    - bring → brought → brought, 
    - think → thought → thought
- **过去式与过去分词不同**：
    - go → went → gone,
    - do → did → done
    - break → broke → broken
- **三形式相同**：
    - cut → cut → cut
    - put → put → put
    - hit → hit → hit
- **以 -en 或 -n 结尾**：
    - eat → ate → eaten 
    - take → took → taken 
    - write → wrote → written
- **元音变化**：
    - 例：drink → drank → drunk, sing → sang → sung, swim → swam → swum

#### NOTICE
- **部分动词有双重形式**：
    - 例：learn → learned（规则，常见于美式）/learnt（不规则，常见于英式）
    - 例：burn → burned/burnt, dream → dreamed/dreamt
- **过去分词作形容词**：
    - 例：broken chair（坏掉的椅子）, written report（写好的报告）
- **发音规律**：
    - 以 /t/ 或 /d/ 结尾的动词，-ed 发 /ɪd/：wanted, needed
    - 以清辅音结尾，-ed 发 /t/：worked, stopped
    - 以元音或其他辅音结尾，-ed 发 /d/：played, lived
