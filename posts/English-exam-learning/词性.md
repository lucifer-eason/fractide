## 🧭 词类（Parts of Speech）

英语中的所有单词，按照“**功能（function）**”与“**性质（nature）**”可以分为以下九大类。  
它们共同构成了英语语言体系的核心。

---

| 类别      | name         | 主要功能        |
| ------- | ------------ | ----------- |
| [[动词]]  | Verb         | 表示动作或状态     |
| [[名词]]  | Noun         | 表示人、物、地点、概念 |
| [[冠词]]  | Article      | 限定名词        |
| [[代词]]  | Pronoun      | 替代名词        |
| [[形容词]] | Adjective    | 修饰名词        |
| [[连词]]  | Conjunction  | 连接成分        |
| [[数词]]  | Numeral      | 表示数量顺序      |
| [[介词]]  | Preposition  | 表示关系        |
| [[感叹词]] | Interjection | 表达情绪        |
