>[!dsnv] 助动词，指==帮助==（所以显然助动词不能单独在句中出现）谓语动词补充一些信息状态。

#### 助动词可通过语法形式分为三类：

##### 1. 基本助动词(这里的助动词，也有（主）动词形式，可单独使用)
>基本助动词并没有特别的实际意义，只付住谓语都动词配置环境/构成关系。

| 助动词  | 功能             | 例句                          | 实义动词                                     |
| ---- | -------------- | --------------------------- | ---------------------------------------- |
| be   | 构成被动关系/配置进行时态。 | He is sleeping.             | To be or not to be, that's the question. |
| have | 辅助构成完成时态。      | He has eaten the hamburger. | I have a pen.                            |
| do   | 添加（肯定/否定）关系    | I don't like you.           | I do well in English.                    |

2. 情态助动词
>情态动词，用来补充谓语动词的情感/状态。

| 情态助动词                  | 功能                        | 例句                                                   |
| ---------------------- | ------------------------- | ---------------------------------------------------- |
| can/ could(过去有能力)      | 有有能力                      | I can finish the homework.                           |
| can/could(更加礼貌）        | 请求                        | Could you help me?                                   |
| can/could(过去的推测/可能性更小) | 可能                        | Anything could happen.                               |
|                        |                           |                                                      |
| may/might(更礼貌）         | 请求                        | Might I get some money from you?                     |
| may/might(可能性更小)       | 可能性（但可能性小于can/could）      | He may be at home.                                   |
| may                    | 表示祝愿                      | =May you succeed!                                    |
|                        |                           |                                                      |
| must                   | 表示必须                      | You must finish you homework on time.                |
| must not               | 表示禁止                      | You must not smoke here.                             |
| must                   | 表推测(肯定)                   | The light is on. He must be at home.                 |
|                        |                           |                                                      |
| will/would(过去将来)       | 配置将来时态                    | I will eat the pizza.                                |
| will/would(更礼貌)        | 表请求                       | Would you please help me?                            |
| will/would(可能性更小)      | 表示推测                      | Ask him. He will know.                               |
|                        |                           |                                                      |
| shall                  | 配置将来时（与will有细微区别详见时态中的使用） | I shall be here at 8am.                              |
| shall                  | 征求意见(请求)                  | Shall we meet here at 8 am?                          |
| shall                  | 表示警告命令                    | You shall not pass.                                  |
| should                 | 表示应当                      | You should treat him well.                           |
| should                 | 表示有一定根据的推测                | It should rain tomorrow.                             |
| should                 | 表竟然                       | That he should be late, it's surprise                |
|                        |                           |                                                      |
| ought to               | 表示应该（但语气比should更强）        | You ought to treat your kid well.                    |
| need                   | 需要                        | The rabbit need wait.                                |
| 易混淆                    | 也可直接做实义动词                 | The rabbit needs to wait.                            |
| dare                   | 敢                         | I dare not tell him the truth.                       |
| used to                | 过去习惯,而现在不。                | I used to play basketball, I only play football now. |
| 易混淆be/get use to       | 习惯做某事。                    | He is used to eating a apple everyday.               |
>[!nucnj] 其中should/would/could/might 也用于辅助配置虚拟语气,其中有不同含义，详见[[虚拟语气（subjunctive mood）]]

>[!njieo] 总结主要功能，差异：
>| 功能类别      | 语气强弱或礼貌程度对比                                                         |
| --------- | ------------------------------------------------------------------- |
| **请求**    | might ＞ may ＞ could ＞ can （从最礼貌到最直接）                                |
| **推测**  [^1]  | must（肯定推测） ＞ will（较大可能） ＞ should（有根据）> can/could ＞ may/might（一般） |
| **建议/义务** | must（必须） ＞ ought to ＞ should（应当） ＞ need（视情况）                        |
| **未来表达**  | shall（正式/英式） ≈ will（常用）                                             |
| **习惯表达**  | used to（过去习惯） ≠ be used to（现在习惯）                                    |

3. 半助动词
>常以词组形式出现，帮助补充更多信息也有更多限制。

| 半助动词                                             | 功能                   | 例句                                                |
| ------------------------------------------------ | -------------------- | ------------------------------------------------- |
| be able to                                       | 有能力（相对can它可表示的时态更自由） | I wil be able to reach school on time.            |
| be (about==马上就要做某事==/going==有计划性==) to==注定要做某事== | 将要做某事                | I think they are to fall in love with each other. |
| had better                                       | 做好做（忠告）              | You‘d better follow my orders.                    |

[^1]: 一般情态动词过去形式表示（推测的可能性更小）
