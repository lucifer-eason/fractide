###   Tense（时间）
1. 过去
	表示 **已经发生的动作或状态**。
	ex：我**昨天去了商店**。
	
2. 现在
	表示 **现在==正在发生==、==经常发生==、或==普遍真理==的状态或动作**。
	ex：
	- 我**现在在学校**。
    - 他**每天读书**。
    - 太阳**从东边升起**。
3. 将来
	表示 **将在“说话之后”的某个时间发生的动作或状态**。
	ex：我**明天要去学校**。
4. 过去将来 
	表示 **“站在过去的角度看未来”** 的事情。
	ex：
	- 他说他**第二天要去北京**。
    - 她以为他**会来**。

###  Aspect（状态）
1. 一般
	表示 **动作的事实、习惯或经常性**，不强调过程或结果。
	ex： 我每天早上**跑步**。
2. 进行
	表示 **动作正在进行或持续发生中**，强调「过程」。
	ex：我**正在吃饭**。
3. 完成
	表示 **动作已经完成，对现在或某一时间有结果或影响。**
	ex：我**已经吃完饭了**。
4. 完成进行
	表示 **动作从过去持续到某一时刻,且还未完全结束**。
	ex：我**已经学习了两个小时**（还在学）。

### 三，然而这依旧无法表示，必须，可能等，为了补充这些功能：[[助动词（auxilary verbs）]]

#### EX:   eat（predicated verb）

| 功能    | 助动词   | 使用格式       |
| ----- | ----- | ---------- |
| 完成    | have  | have eaten |
| 进行    | be    | be eating  |
| 被动    | be    | be eaten   |
| 有能力   | can   | can eat    |
| 有义务   | must  | must eat   |
| 有可能   | might | might eat  |
| （是/否） | do    | do（not）eat |

> [!tip] Notice
> 助动词，常有其它词性，以及意思，单独使用。
